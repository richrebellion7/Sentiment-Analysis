# Bilingual Sentiment Analysis using Transformers

## Overview

This project performs sentiment analysis on bilingual English + Hinglish movie reviews using transformer-based NLP models.

## Features

* Multilingual sentiment classification
* Hinglish and noisy text preprocessing
* Emoji normalization
* Weak supervision for dataset labeling
* Transformer fine-tuning with XLM-RoBERTa

## Tech Stack

* Python
* HuggingFace Transformers
* XLM-RoBERTa
* Google Colab
* Pandas
* Scikit-learn

## Future Improvements

* Larger multilingual dataset
* Real-time inference API
* Improved slang and dialect understanding
